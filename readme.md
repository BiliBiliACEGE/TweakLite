# TweakLite

TweakLite is a lightweight Fabric mod offering configurable client and server tweaks for Minecraft.

## Features

- **Creative Tools**: Copy entity NBT data to spawn eggs with Ctrl+Middle Click
- **Player Utilities**: Adjustable attack range, block reach, and player scale
- **Entity Management**: Scale entities and shear goat horns
- **Visual Enhancements**: Remove pumpkin blur overlay
- **In-game Config**: Press **X + K** (default) to open the configuration GUI

## Configuration

The mod includes an in-game configuration interface for easy customization.

- **Open Config GUI**: `X + K` (default keybind)
- All tweaks can be toggled and customized through the config screen
